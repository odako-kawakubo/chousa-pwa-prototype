# 調査システムPWA プロトタイプ

調査システムPWA化PJの触れる簡易プロトタイプです。

## GitHub Pagesで公開する手順

1. GitHubで新規リポジトリを作成
2. このフォルダ内の `index.html` と `README.md` をアップロード
3. リポジトリの Settings > Pages を開く
4. Branch を `main`、Folder を `/root` にして Save
5. 数分後に表示されるURLへアクセス

公開URLの形式は通常以下です。

https://ユーザー名.github.io/リポジトリ名/

## 内容

- 案件画面
- 仕上表入力
- 建材登録ボタン
- 建材リスト反映
- 写真追加
- 試料画面
- 同期状態
